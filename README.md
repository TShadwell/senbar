###Senbar

A taskbar for i3.

To get it, do this:

`go get -u github.com/TShadwell/senbar/senbar`

###The Interesting Bits
This project also includes a pretty good, but incomplete i3 library, as well as a native golang implimentation of some dzen gadgets. All docs can be found [here](http://go.pkgdoc.org/github.com/TShadwell/senbar).
